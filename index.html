Sistema de Gestión de reclutamiento de talentos

Este módulo de ERP combina tecnologías web modernas y machine learning. Utiliza Python y Flask como base para el backend, integrando modelos predictivos con scikit-learn para mejorar la selección de candidatos mediante análisis de datos históricos y características de perfiles exitosos. La interfaz está construida con HTML5, CSS3 y JavaScript, ofreciendo una experiencia de usuario ágil e intuitiva. El sistema almacena y gestiona la información en una base de datos SQLite, administrada mediante SQLAlchemy, lo que permite consultas eficientes y persistencia de datos estructurada.

¡Hola! Soy Felipe Bianciotto

Soy estudiante de la Tecnicatura Universitaria en Programación con un especial interés en la calidad del software. Mi objetivo es desarrollar soluciones robustas, mantenibles y escalables, aplicando las mejores prácticas de programación.

Desde que comencé mis estudios, me he enfocado en entender no solo cómo crear código funcional, sino también cómo asegurar su calidad y facilitar su mantenimiento a largo plazo. Creo firmemente que estos aspectos deben ser considerados desde el inicio de cualquier proyecto.

Mi enfoque se centra en el aprendizaje continuo, la atención al detalle y la implementación de soluciones que no solo funcionen, sino que también sean confiables y escalables.
